VideoProjection
===============

Video projection system

--- Keyboard Command ---

Projection adjustment:

F1: help

F2, F3, F4, F5: Axes

F9, F10: -/+ Alpha Mask

F12: Save

--- By configuration ---

Color tint:

1: white
2: Red
3: Green
4: Blue
5: Yellow
6: Violet
7: Azure

Video selection:

z: none
x: video_1
c: video_2
